# 🤖 AI 퀀트 트레이딩 시스템 사용 가이드

본 시스템은 **32비트(데이터/주문)**와 **64비트(AI 분석)** 가상환경으로 철저히 분리된 하이브리드 아키텍처로 구동됩니다. 매일 주식 시장이 열리기 전이나 장 중에 아래의 순서대로 스크립트를 실행해 주시면 됩니다.

---

## 📅 매일 아침 구동 루틴 (Daily Workflow)

### 1단계: 시장 데이터 및 테마 수집 (데이터 준비)
아침에 제일 먼저 해야 할 일은 **최신 데이터**를 가져오는 것입니다. 명령 프롬프트(CMD)나 VSCode 터미널에서 다음을 실행하세요.

```bash
# 1. 주도 테마 100종목 추출 (32비트)
.\venv32\Scripts\python.exe theme_tracker.py

# 2. 전 종목 최신 일봉 차트 스크래핑 (32비트)
.\venv32\Scripts\python.exe screener.py
```
> **동작 확인:** 키움증권 로그인 창이 뜨고 모의투자로 로그인하면, 알아서 데이터를 수집하고 `kiwoom_data.db`에 저장한 뒤 종료됩니다.

### 2단계: 자동 주문 사령관 띄우기 (실행/방어 대기)
이제 실제 주문을 쏘고 계좌를 방어할 **Order Manager**를 켜둘 차례입니다. 이 창은 장이 끝날 때까지 끄면 안 됩니다.

```bash
# 3. 자동 주문 시스템 구동 (32비트)
.\venv32\Scripts\python.exe order_manager.py
```
> **동작 확인:** 로그인 후 `[OrderManager] DB 큐 모니터링을 시작합니다.` 라는 문구가 뜨고 3초마다 DB를 감시하며 대기합니다.

### 3단계: AI 전략 엔진 가동 (타점 포착 및 신호 발송)
이제 64비트 AI 뇌를 가동하여 타점을 계산합니다. 장 중에 원할 때마다 이 스크립트를 실행하거나, 윈도우 스케줄러를 통해 30분 단위로 실행하게 만들면 알아서 신호를 보냅니다.

```bash
# 4. AI 퀀트 엔진 가동 (64비트)
.\venv64\Scripts\python.exe strategy_engine.py
```
> **동작 확인:** 
> 1. 매크로 국면 분석기가 현재 시장(상승/하락/횡보)을 분석합니다.
> 2. `strategy_engine.py`가 MACD 영선 돌파 등 A급 타점을 찾아냅니다.
> 3. 타점을 찾으면 화면에 알림이 뜨고, 켜두었던 `order_manager.py`가 이를 0.1초 만에 낚아채어 **즉시 키움증권 모의투자로 시장가 매수**를 때립니다! 🚀

## 🛡️ 모니터링 및 킬 스위치 (알아서 동작함)
- **트레일링 스탑:** `order_manager.py`가 켜져 있는 동안, 방금 산 종목이 오르면 최고가를 기억합니다. 만약 최고가 대비 **-3%** 이상 하락하면 알아서 전량 매도 쳐서 수익을 방어합니다. 신경 쓰지 않으셔도 됩니다!
- **킬 스위치:** 만약 오늘 손실액이 10만 원을 넘어가면 콘솔에 해골 마크(💀)가 뜨며 더 이상 매수 신호를 받지 않습니다. HTS나 MTS를 켜서 계좌 상태를 확인해 주세요.

## 💡 팁 (배치 파일 활용)
매번 타이핑하기 귀찮으시다면, 메모장을 열고 위 명령어들을 순서대로 적은 뒤 `run.bat` 같은 이름으로 저장하여 더블 클릭 한 번으로 실행되게 만들 수 있습니다.
